## Simple CRUD API in nodejs

## Running the project

run,

- `npm install`
- `npm run dev`

server should be running at: `http://localhost:3000/`

## Sample payload

`{
    "user_id": "user_id",
    "products": {
        "product_id_1": 1,
        "product_id_2": 1
    },
    "payment_info": "Payment details"
}`
